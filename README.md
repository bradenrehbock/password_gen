# Password Generator

## Description
This project is a simple web-based password generator. It allows the user to choose a password length and decide whether or not to include numbers in the generated password. The result is displayed on the page and can be copied with the click of a button.

## How To Use
1. Open the website in your browser.
2. Enter your desired password length.
3. Check the box if you want numbers included.
4. Click **Generate Password**.
5. Click **Copy Password** to copy the generated password to your clipboard.

## Example Output
A generated password may include lowercase letters, uppercase letters, symbols, and optionally numbers. For example:

`U;IRDbfWiv)Q`
