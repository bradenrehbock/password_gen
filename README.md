# Password Generator

## Description
This project allows an individual to generate a randomized password. 
They have the option to choose the length, and whether or not they 
want numbers included in what is generated.

## How To Use
Run the script by calling the function with two inputs:
- Your desired password length
- Whether you want numbers included (True or False)

Example:
gen_password(12, True)

## Example Output
A generated password can contain any combination of lowercase letters, 
uppercase letters, symbols, and numbers. For example:
L0129dj!*n

## Built With
- Python
- secrets module
- string module
